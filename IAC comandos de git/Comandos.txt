ls:lista diretorios e arquivos de uma pasta
cd .. é para voltar um nível de diretório ou pasta

clear: limpar tela
mkdir: para criar pastas
git add: adicionar as modificações na area de commit

git commit -m "mensagem do commit":esse comando etiqueta os arquivos para enviar ao repositório remoto
git push: envia os commits para o repositório remoto
